A simple music bar is used to control a 1g1g player, show the lyric of the playing song.
亦歌音乐栏，控制亦歌播放，显示当前歌词。

== Description ==

The widget can be installed in the sidebar of a blog powered by Wordpress. While the blog is loaded, the widget can open or control a 1g1g player and show the lyric of the playing song.
此小工具可安装在由WordPress支持的博客系统的侧边栏中。用户浏览博客时，显示亦歌播放歌曲的当前歌词。并且，它可以简单地控制亦歌的播放进程。

== Installation ==

* Install directly from WordPress (just go to *Plugins* -> *Add New*) and activate.
Or
* Manually install to your WP plugins directory
	1. Unzip `1g1g-music-bar.zip` in the `/wp-content/plugins/` directory.
	2. Activate the plugin through the *Plugins* menu in WordPress.
	3. Select a favorite color of the widget in the widgets admin page.
	
[安装方法]
* 在管理面板中选择“插件”->“添加新插件”，搜索“1g1g”，看到“1g1g Music Bar Widget”，点击“安装”。在弹出页面的右上角点击“现在安装”。安装完成后点击“激活插件”。
* 在“外观”->“小工具”中，看到“Music bar from 1g1g”，将它拖拽至右侧边栏中您希望的位置。点击小工具右方的下拉箭头，可以对其进行配置，当前提供“初始文字”、“背景颜色”等配置选项。

== Frequently Asked Questions ==

= Is the 1g1g web site opened firstly? =
No, the music bar will detect 1g1g web automatically. If the web hasn't been opened, the music bar will open a new web page of the 1g1g site, when the start button is clicked.
