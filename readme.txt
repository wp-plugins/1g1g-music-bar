A simple music bar is used to control a 1g1g player, show the lyric of the playing song.

== Description ==

The widget can be installed in the sidebar of a blog powered by Wordpress. While the blog is loaded, the widget can open or control a 1g1g player to play music and show the lyric of the song.

== Installation ==

* Install directly from WordPress (just go to *Plugins* -> *Add New*) and activate.

Or

* Manually install to your WP plugins directory
	1. Unzip `1g1g-musicbar.zip` in the `/wp-content/plugins/` directory.
	2. Activate the plugin through the *Plugins* menu in WordPress.
	3. Select a favorite color of the widget in the widgets admin page.

== Frequently Asked Questions ==

= Is the 1g1g web site opened firstly? =
No, the music bar will detect 1g1g web automatically. If the web hasn't been opened, the music bar will open a new web page of the 1g1g site, when the start button clicked.
